# Kanban Board (Учебная практика)

Полноценное веб-приложение для управления задачами по методологии Kanban. Разработано в рамках учебной практики.

## Реализованный функционал (MVP)

- **Авторизация и аутентификация**: Регистрация и вход с использованием JWT-токенов и хэшированием паролей (bcrypt).
- **Ролевая модель доступа (RBAC)**: 
  - **Владелец**: полный доступ, управление участниками и их ролями.
  - **Редактор**: создание и редактирование досок, колонок, карточек и комментариев.
  - **Читатель**: только просмотр доски и перемещение карточек.
- **Управление досками**: создание, редактирование, удаление, переключение между досками.
- **Колонки и Карточки**: создание, удаление, редактирование (название, описание, приоритет, дедлайн, исполнитель).
- **Drag-and-Drop**: интуитивное перемещение карточек между колонками с мгновенным сохранением в БД.
- **Комментарии**: обсуждение задач внутри карточек.
- **Защита от коллизий**: механизм оптимистичных блокировок (поле `version`) предотвращает перезапись данных при одновременном редактировании.
- **Audit Log**: полная история изменений карточек сохраняется в таблице `card_history`.

## Технологический стек

**Backend:**
- Python 3.10+
- FastAPI (асинхронный фреймwork)
- SQLAlchemy 2.0 (ORM)
- SQLite (для локальной разработки, архитектура совместима с PostgreSQL)
- Pydantic (валидация данных)
- Passlib / Bcrypt (хэширование паролей)
- Python-Jose (JWT)

**Frontend:**
- React 18 + Vite
- @hello-pangea/dnd (Drag-and-Drop)
- React Router DOM (маршрутизация)
- CSS (лаконичный минималистичный дизайн в стиле Tiffany)

## Инструкция по запуску

### Требования
- Python 3.10 или выше
- Node.js 18 или выше

### 1. Запуск Backend
Откройте терминал и выполните следующие команды:
```bash
# Перейдите в папку backend
cd backend

# Создайте и активируйте виртуальное окружение (Windows)
python -m venv venv
venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt

# Наполните базу данных тестовыми данными
python seed_data.py

# Запустите сервер разработки
uvicorn app.main:app --reload --port 8001
```
Swagger-документация API будет доступна по адресу: http://127.0.0.1:8001/docs

### 2. Запуск Frontend
Откройте **новый** терминал и выполните:
```bash
# Перейдите в папку frontend
cd frontend

# Установите зависимости
npm install

# Запустите сервер разработки
npm run dev
```
Приложение будет доступно по адресу: http://localhost:5173 (или 5174)
